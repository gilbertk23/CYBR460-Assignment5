# CYBR460-Assignment5
This GitHub repository contains the files for my CYBR 460 Assignment 5 (Containerizing Applications with Docker) submission. The files can be located with [this link.](https://github.com/gilbertk23/CYBR460-Assignment5/blob/main/CYBR460Assignment5.zip) 
Once on this page, you can press the "download raw file" button on the righthand side of the page to download a .zip folder containing four files:
- A README file with instructions on building the Docker image and how to run the container to produce the program's expected output.
- A .py file cntaining the source code of the program.
- A Dockerfile for containerizing the program.
- A .txt file containing a demonstration of the program's results.
